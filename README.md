# Slide decks

Repositório de apresentações de **Fabrízio Krapf Costa** (fabrizio@mirador360.com.br).

As apresentações ficam disponíveis em: **https://fkrapf.github.io/slide_decks/**

## Acesso às apresentações

O repositório é publicado **sem um índice** (sem página inicial). Isso significa que, para acessar uma apresentação específica, é necessário **conhecer o nome do diretório** correspondente e acessá-lo diretamente pela URL.

Por exemplo:

```
https://fkrapf.github.io/slide_decks/<nome_do_diretorio>/
```

Cada diretório contém uma apresentação. Consulte o `README.md` de cada diretório para detalhes sobre a apresentação e instruções de publicação.
